# 🏥 Medical Diagnosis Assistant

An **AI-powered medical diagnosis assistant** that analyzes user symptoms written in natural language and predicts possible diseases with confidence scores.

The system combines **Natural Language Processing (NLP), rule-based reasoning, and data-driven matching** to simulate a simple medical diagnostic assistant.

⚠️ **Disclaimer:** This project is for educational purposes only and should **not be used for real medical diagnosis**.

---

# 🚀 Features

- Convert natural language symptoms into structured data
- Detect symptoms using **NLP (NLTK)**
- Map symptoms to diseases using a **knowledge-based system**
- Calculate **diagnosis confidence scores**
- Ask **follow-up questions** to improve prediction accuracy
- Display **possible diseases with precautions**

---

# ⚙️ Technologies Used

- **Python**
- **NLTK** (Natural Language Processing)
- **Experta** (Rule-Based Expert System)
- **NetworkX** (Graph relationships)
- **Pandas**
- **NumPy**
